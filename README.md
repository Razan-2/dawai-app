# دوائي (Dawaee) - تطبيق تذكير الأدوية 💊

تطبيق ويب متكامل يهدف إلى مساعدة المرضى وكبار السن في تنظيم ومتابعة مواعيد أدويتهم، وتقليل التعارض الدوائي، وتسهيل العثور على الصيدليات القريبة من خلال تفاعل حي مع خريطة جغرافية واضحة وبسيطة.

## 🚀 المميزات الرئيسية
1. **نظام المصادقة الشامل:** من خلال `Supabase Auth` (تسجيل دخول، إنشاء حساب جديد، عزل بيانات المستخدمين).
2. **جدولة وتنبيهات الأدوية:** إدخال اسم الدواء، النوع، مواعيد الأخذ، وتنبيهات صوتية أو بصرية عبر المتصفح (Notification API).
3. **فحص التفاعلات الدوائية:** أداة لمحاكاة فحص التعارضات بين دوائين للحفاظ على سلامة المريض.
4. **خريطة الصيدليات التفاعلية:** مبنية على مكتبة `Leaflet` تحسب مسافة المستخدم عن أقرب الصيدليات (باستخدام خوارزمية Haversine) وترتيبها حسب القرب مع إمكانية التحويل لتطبيق الخرائط للتوجيه.
5. **لوحة تحكم وتصميم عصري:** تصميم (Glassmorphism) مريح للعين يدعم (الوضع الداكن/الفاتح) ومراعٍ للغات من اليمين لليسار (RTL).

## 🛠️ كيف يعمل التطبيق / التقنيات المستخدمة
- **الواجهة الأمامية (Frontend):** HTML, CSS (Vanilla), JavaScript
- **الخريطة التفاعلية:** مكتبة [Leaflet.js](https://leafletjs.com) وخريطة `OpenStreetMap` للصور الخرائطية (Tiles).
- **الخلفية وقواعد البيانات (Backend Database):** [Supabase](https://supabase.com/) لإدارة جداول المستخدمين (`users`), الأدوية (`medicines`), والتنبيهات (`reminders`). 
- **مكتبة الأيقونات الجمالية:** [Font Awesome 6](https://fontawesome.com/).

## 📁 هيكلية المشروع (ملفات المصدر الأساسية)
يتبع المشروع نمطاً بسيطاً نظيفاً مناسب للنشر المباشر على خوادم الويب:
- `index.html`: الصفحة الرئيسية (Dashboard) التي تحوي الخريطة وقائمة الأدوية والإعدادات.
- `login.html`: صفحة تسجيل الدخول.
- `register.html`: صفحة إنشاء حساب مستخدم جديد.
- `style.css`: ملف التنسيق الموّحد لجميع الصفحات.
- `app.js`: الملف البرمجي الأساسي للوحة التحكم، إدارة الخريطة، وجلب بيانات الأدوية.
- `auth.js`: الملف البرمجي الخاص بإدارة مصادقة المستخدمين في Supabase.
- `supabase_schema.sql`: أكواد الإنشاء (SQL Queries) لجداول وقواعد البيانات.

## ⚙️ التثبيت والنشر خطوة بخطوة 
التطبيق من نوع "تطبيق ويب من صفحة واحدة" (SPA-like) ويعمل مباشرة على المتصفح:

1. **الاستنساخ (Clone):**
   ```bash
   git clone https://github.com/YOUR_GITHUB_USERNAME/Dawaee-App.git
   ```
2. **التهيئة (Supabase Setup):**
   - قم بالتسجيل في [Supabase](https://supabase.com) وإنشاء مشروع جديد.
   - افتح نافذة SQL Editor وانسخ والصق محتوى `supabase_schema.sql` لبناء الجداول.
   - قم بنسخ `URL` المشورع و `Anon Key` ولصقهما في أعلى ملفّي `app.js` و `auth.js`.
3. **التشغيل محلياً:** يمكنك فتح `login.html` في متصفحك مباشرة أو استخدام إضافة `Live Server` في VS Code.

## 🌐 النشر على Vercel أو GitHub Pages
هذا التطبيق مكوّن بالكامل من ملفات ثابتة (Static HTML/CSS/JS)، لذلك فهو مناسب بنسبة 100% للرفع المباشر:

- **عبر Vercel:** قم بتسجيل الدخول، إنشاء "New Project"، قم باختيار مستودع هذا المشروع من مساحة عملك على GitHub وسيقوم Vercel تلقائيا، ودون أي إعدادات معقدة بتهيئة روابط وشهادات الأمان ونشر الموقع فورا.
- **عبر GitHub Pages:** اذهب إلى إعدادات المستودع -> `Pages` -> قم بتحويل المصدر إلى `main branch` ثم اضغط حفظ. وسيظهر الرابط بعد لحظات.
